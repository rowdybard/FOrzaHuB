Landing hero media
==================

Drop the hero drift loop here with these names:

- landing-drift-loop.mp4
- landing-drift-loop.webm, optional
- landing-drift-poster.jpg, optional poster frame

Recommended CapCut export:

- MP4, H.264
- 1920x1080
- 6 to 10 seconds
- 24 or 30 fps
- muted/no audio
- under 8 MB if possible

Use MP4/WebM instead of GIF for the hero background. It will load smoother,
look cleaner, and stay much smaller.
